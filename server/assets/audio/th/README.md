# ไฟล์เสียงเรียกคิว (ภาษาไทย)

วางไฟล์เสียง `.wav` (หรือ `.mp3`) ในโฟลเดอร์นี้ ระบบจอแสดงผลจะนำมาเล่นต่อกันเป็นประโยค
ตัวอย่างประโยค: **"เชิญหมายเลข L60801 ที่ห้องฉายรังสี L6 ค่ะ"**

ประโยคถูกต่อจาก 3 ส่วน: `please_number` + สะกดเลขคิวทีละตัว + ไฟล์ห้อง

## ไฟล์ที่ต้องมี (ชื่อไฟล์ = คีย์)
| ไฟล์ | เสียงพูด |
|------|----------|
| `please_number.wav` | "เชิญหมายเลข" |
| `0.wav` ... `9.wav` | ตัวเลข ศูนย์–เก้า |
| `L.wav` | ตัวอักษร "แอล" |
| `L6_room.wav` | "ที่ห้องฉายรังสี L6 ค่ะ" (หางประโยคครบ รวม "ค่ะ") |
| `L8_room.wav` | "ที่ห้องฉายรังสี L8 ค่ะ" |

> คีย์ตัวอักษร/ตัวเลขมาจากการสะกดเลขคิวทีละตัว เช่น `L60801` → `L 6 0 8 0 1`
> ชื่อไฟล์ห้องมาจากฟิลด์ `voice_room_key` ของแต่ละห้อง (ตั้งค่าในหน้า Settings) — ค่าเริ่มต้น `L6_room`, `L8_room`
> ไฟล์ห้องต้องอัด **"ที่ห้อง...ค่ะ" ครบทั้งหาง** เพราะระบบไม่ได้ต่อ "ที่ห้อง" / "ค่ะ" แยกอีกแล้ว

## ไฟล์เสริม (ใช้ที่ kiosk — ยังไม่ผูกอัตโนมัติ)
| ไฟล์ | เสียงพูด |
|------|----------|
| `choose_time.wav` | "เลือกช่วงเวลา" |
| `please_take_ticket.wav` | "กรุณารับบัตร" |

## หมายเหตุ
- ระบบจะลองโหลด `.wav` ก่อน ถ้าไม่เจอจึงลอง `.mp3`
- หากไฟล์เสียงใดขาดหาย ระบบจะ **fallback ไปใช้เสียงสังเคราะห์ (TTS) ภาษาไทยของเบราว์เซอร์** ทั้งประโยคโดยอัตโนมัติ
- แนะนำอัดเสียงด้วยน้ำเสียงเดียวกันทุกไฟล์ ความดังใกล้เคียงกัน ตัดช่องว่างหัว-ท้ายไฟล์ออก
- ต้นฉบับไฟล์เสียงเก็บไว้ที่โฟลเดอร์ `voice/` ที่รากโปรเจกต์ (ก๊อปมาที่นี่เพื่อให้เซิร์ฟเวอร์เสิร์ฟ)
