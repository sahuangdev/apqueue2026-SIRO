# วิธีตั้งค่าเครื่องจอแสดงผลคิว (Display)

คู่มือสำหรับติดตั้ง **เครื่องที่ใช้แสดงจอคิว** (จอทีวี/มอนิเตอร์ที่เปิดหน้า `…/display/`)
ให้แสดงผลเต็มจอ และ **เล่นเสียงเรียกคิวอัตโนมัติโดยไม่ต้องแตะหน้าจอ**

> เครื่องจอแสดงผลมักเป็นคนละเครื่องกับ **server** (เครื่องที่รันโปรแกรมหลัก/ฐานข้อมูล)
> คู่มือนี้ตั้งค่าที่ **เครื่องจอ** เท่านั้น

---

## ภาพรวม: อะไรอยู่ที่ไหน

| สิ่งที่ตั้งค่า | ตั้งที่ไหน | ต้องทำซ้ำทุกเครื่องจอไหม |
|---|---|---|
| เอาปุ่ม "แตะเพื่อเปิดเสียง" ออก / โค้ดปลดล็อกเสียง | อยู่ที่ **server** (เสิร์ฟไฟล์เว็บ) | ❌ ไม่ต้อง — ทุกเครื่องได้อัตโนมัติ |
| ให้เสียงเล่นทันที (ไม่ต้องแตะ) | ที่ **เครื่องจอ** (วิธีเปิดเบราว์เซอร์) | ✅ ต้องตั้งทุกเครื่องจอ |

**เหตุผล:** Chrome/Edge ปกติจะ *บล็อกการเล่นเสียงอัตโนมัติ* จนกว่าจะมีคนคลิก/แตะหน้าจอครั้งแรก
เป็นกฎความปลอดภัยของเบราว์เซอร์เอง แก้ในโค้ดเว็บล้วน ๆ ไม่ได้
จึงต้องเปิดเบราว์เซอร์ด้วย flag พิเศษ `--autoplay-policy=no-user-gesture-required`
ซึ่งทำให้ไว้แล้วในไฟล์ `start-display.cmd`

---

## ขั้นตอนติดตั้งที่เครื่องจอ

### 1. ติดตั้ง Google Chrome
ดาวน์โหลดและติดตั้งจาก https://www.google.com/chrome
(ถ้าไม่มี Chrome ไฟล์จะใช้ Microsoft Edge ที่ติดมากับ Windows แทนให้อัตโนมัติ)

### 2. คัดลอกไฟล์ `start-display.cmd` มาที่เครื่องจอ
วางไว้ที่ไหนก็ได้ เช่น Desktop

### 3. แก้ URL ให้ชี้ไปที่ server
เปิดไฟล์ `start-display.cmd` ด้วย Notepad แล้วแก้บรรทัด:

```bat
set "URL=http://localhost:8888/display/"
```

- ถ้าเครื่องจอ **เป็นเครื่องเดียวกับ server** → ใช้ `localhost` ได้เลย ไม่ต้องแก้
- ถ้าเครื่องจอ **เป็นคนละเครื่อง** → เปลี่ยน `localhost` เป็น **IP ของเครื่อง server** เช่น

```bat
set "URL=http://192.168.1.50:8888/display/"
```

> **หา IP เครื่อง server ยังไง:** ไปที่เครื่อง server เปิด Command Prompt พิมพ์ `ipconfig`
> ดูค่า **IPv4 Address** (เช่น 192.168.1.50) ของการ์ดเน็ตที่ใช้งานอยู่

### 4. เปิดใช้งาน
ดับเบิลคลิก `start-display.cmd` → Chrome จะเปิดเต็มจอ และเสียงเรียกคิวเล่นได้ทันที

- **ออกจากโหมดเต็มจอ:** กด `Alt + F4`

---

## ตั้งให้เปิดอัตโนมัติตอนเปิดเครื่อง (แนะนำ)

เพื่อไม่ต้องเปิดเองทุกครั้งหลังเปิด/รีสตาร์ทเครื่อง:

1. กด `Win + R` พิมพ์ `shell:startup` แล้ว Enter → จะเปิดโฟลเดอร์ Startup
2. คลิกขวาที่ `start-display.cmd` → **Create shortcut** (สร้างทางลัด)
3. ลากไฟล์ทางลัดที่ได้ ไปวางในโฟลเดอร์ Startup

ครั้งต่อไปเมื่อเปิดเครื่อง จอคิวจะเปิดเต็มจอ + เสียงพร้อมใช้งานเองอัตโนมัติ

---

## แก้ปัญหาเบื้องต้น

| อาการ | สาเหตุ / วิธีแก้ |
|---|---|
| เปิดแล้วหน้าจอว่าง/ต่อไม่ติด | URL ผิด หรือ server ยังไม่เปิด — ตรวจ IP server และให้แน่ใจว่า server รันอยู่ |
| ภาพขึ้นแต่ **ไม่มีเสียง** | ไม่ได้เปิดผ่าน `start-display.cmd` (เปิดแท็บ Chrome ธรรมดา) — ปิดแล้วเปิดใหม่ด้วยไฟล์นี้ |
| ยังไม่มีเสียงแม้เปิดด้วยไฟล์ | ลองลบโฟลเดอร์โปรไฟล์ `%LOCALAPPDATA%\SiroDisplay` แล้วเปิดใหม่ / ตรวจว่าลำโพงเครื่องไม่ได้ปิดเสียง |
| เห็นจอเก่า/ยังมีปุ่มเปิดเสียง | กด `Ctrl + F5` รีเฟรชล้างแคชหนึ่งครั้ง |
| สองเครื่องจอ ต้องแสดงคนละห้อง | หน้านี้แสดงทุกห้องเหมือนกันทุกเครื่อง (ออกแบบไว้แบบนั้น) ถ้าต้องการแยกห้องต่อจอ แจ้งผู้พัฒนาเพิ่ม |

---

## สรุปสั้น ๆ

> เครื่องจอใหม่ทุกเครื่อง: **คัดลอก `start-display.cmd` → แก้ URL เป็น IP server → เปิดด้วยไฟล์นี้ → (ทางเลือก) ใส่ Startup**
> เท่านี้ก็แสดงผลเต็มจอ + เสียงเรียกคิวเล่นเองโดยไม่ต้องแตะหน้าจอ
